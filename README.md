# 🎬 IMDB Sentiment Analysis using NLP

This project focuses on performing sentiment analysis on movie reviews using Natural Language Processing (NLP) techniques. The dataset used consists of IMDB reviews, with labels indicating whether the sentiment is positive or negative.

---

## 📊 Dataset

- **Name:** IMDB Sentiment Dataset
- **Original Size:** 100,000 movie reviews
- **Used Size:** Reduced to 10,000 reviews for convenience
- **Source:** [Kaggle - IMDB Sentiments](https://www.kaggle.com/datasets/jcblaise/imdb-sentiments)

---

## 📌 Project Description

Natural Language Processing (NLP) enables computers to understand, interpret, and generate human language. This project utilizes NLP for:

- Sentiment analysis
- Text classification

### Workflow:

1. **Text Preprocessing**:
   - Lowercasing
   - Removing punctuation and special characters
   - Stop word removal
   - Tokenization
   - Lemmatization

2. **Modeling Techniques**:
   - Vectorization using TF-IDF
   - Model training using machine learning algorithms (e.g., Logistic Regression, Naive Bayes)

3. **Evaluation**:
   - Accuracy, Precision, Recall, F1-score
